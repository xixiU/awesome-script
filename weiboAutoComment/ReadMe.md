简介:微博批量自动评论<br>

说明：
搜索特定关键词之后，运行本脚本，可以批量自动评论(自定义评价内容)并翻页。

<strong style="color: red">请大家文明评论，本脚本仅供学习交流使用，请勿用于其他用途，本人不对此负任何责任。</strong>
