# 字幕功能快速参考卡

## 一分钟上手

### 1. 启动后端

```bash
cd subtitle_backend && ./start.sh
```

### 2. 使用字幕

- 访问视频网站
- 按 **S 键** 开启字幕
- 完成！

## 常用操作

| 操作 | 方法 |
|------|------|
| 开启字幕 | 按 S 键 或 点击 📝 按钮 |
| 关闭字幕 | 再按 S 键 或 再点击按钮 |
| 配置 | 油猴菜单 → 字幕翻译配置 |
| 重启 | 油猴菜单 → 重启字幕服务 |

## 配置

### 默认配置

- 服务地址：`http://localhost:8765`
- 目标语言：`zh-CN`（简体中文）
- 自动翻译：开启

### 修改语言

油猴菜单 → 字幕翻译配置 → 输入语言代码

常用语言代码：

- `zh-CN` - 中文
- `en` - 英语
- `ja` - 日语
- `ko` - 韩语

## 快捷键

| 键 | 功能 |
|----|------|
| S | 字幕开关 |
| V | 加速 |
| X | 减速 |
| P | 截图 |
| I | 画中画 |

## 故障排查

**字幕不显示？**
→ 检查后端是否运行：`curl http://localhost:8765/health`

**识别不准？**
→ 后端使用更大模型（small/medium）

**有延迟？**
→ 正常现象，5-10秒延迟

## 提示

- ✅ 确保视频音质清晰
- ✅ 使用有线网络
- ✅ 关闭其他资源占用程序
- ✅ Chrome/Edge 浏览器效果最佳
