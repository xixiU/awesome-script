# HTML5视频播放工具

## 功能特性

- ✅ 支持 `<video>` 和 `<audio>` 元素倍速控制
- ✅ 可视化倍速UI (拖动滑块 + 预设按钮)
- ✅ 视频截图、画中画、缓存
- ✅ 全屏、网页全屏
- ✅ 字幕服务
- ✅ 万能网页全屏

## 快捷键

### 倍速控制
| 按键 | 功能 |
|------|------|
| **Q** | 显示/隐藏倍速面板 |
| **E** | 加速 +0.1x |
| **C** (YouTube用V) | 加速 +0.1x |
| **X** (抖音用S) | 减速 -0.1x |
| **Z** (抖音用A) | 1.0x ↔ 记忆速度 |

### 其他功能
| 按键 | 功能 |
|------|------|
| **P** | 视频截图 |
| **I** | 画中画 |
| **M** | 缓存视频 |
| **S** | 字幕服务 |
| **N** | 下一集 |
| **F / Enter** | 全屏 |
| **Shift+Enter** | 网页全屏 |
| **←→** | 快进/快退 5秒 |
| **Shift+←→** | 快进/快退 20秒 |
| **↑↓** | 音量调节 |
| **D** | 上一帧 |
| **G** (YouTube用E) | 下一帧 |
| **Space** | 暂停/播放 |
| **ESC** | 退出全屏 |

## 倍速UI使用

### 显示面板
```
按 Q 键
```

### 调节速度
- **预设按钮**: 0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x
- **拖动滑块**: 0.25x - 4.0x 精确调节
- **键盘微调**: E键 +0.1x, X键 -0.1x

### 特性
- 3秒自动隐藏
- 鼠标悬停保持显示
- 实时速度同步

## 常见问题

**Q: 倍速面板不显示?**  
A: 确认页面有视频/音频,按Q键手动显示,或刷新页面

**Q: 音频不支持倍速?**  
A: 确认是HTML5 audio元素,某些Flash播放器不支持

**Q: 快捷键冲突?**  
A: YouTube/抖音已自动重映射快捷键避免冲突
