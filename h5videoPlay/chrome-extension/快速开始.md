# Chrome 扩展快速开始指南

## 🎯 三步安装

### 1. 加载扩展（30秒）

```
打开 Chrome → chrome://extensions/
↓
开启"开发者模式"（右上角）
↓
点击"加载已解压的扩展程序"
↓
选择 chrome-extension 文件夹
↓
✅ 完成！
```

### 2. 启动后端（1分钟）

```bash
cd subtitle_backend
./start.sh
```

等待显示：

```
服务地址: http://localhost:8765
```

### 3. 开始使用（立即）

```
访问 YouTube / B站
↓
播放视频
↓
按 S 键
↓
✅ 字幕显示！
```

## ⚙️ 配置（可选）

点击扩展图标 → 设置：

- 后端地址（默认 localhost:8765）
- 目标语言（默认中文）
- 自动翻译（默认开启）

## ✅ 验证安装

### 检查扩展

```
chrome://extensions/
→ 找到"HTML5视频播放工具"
→ 状态应该是"已启用"
```

### 检查后端

```bash
curl http://localhost:8765/health
# 应返回: {"status":"healthy","model_loaded":true}
```

## 🐛 如果不工作？

### 扩展问题

```
chrome://extensions/
→ 点击"错误"查看详情
→ 或点击"Service Worker"查看后台日志
```

### 字幕问题

```
F12 打开控制台
→ 查看是否有错误
→ 应该看到 [H5Video Extension] 相关日志
```

### 后端问题

```bash
# 重启服务
pkill -f "python server.py"
cd subtitle_backend
./start.sh
```
