# 🚀 快速使用指南

## 一、基础功能（开箱即用）

### 安装

1. 安装 Tampermonkey 浏览器扩展
2. 复制 `html5videoPlay.js` 内容创建新脚本
3. 访问任何视频网站即可使用

### 常用快捷键

| 快捷键 | 功能 | 备注 |
|--------|------|------|
| `V` | 加速 0.1x | YouTube 用 |
| `X` | 减速 0.1x | |
| `Z` | 切换速度 | 1.0 ⇄ 上次速度 |
| `← →` | 快退/快进 5秒 | |
| `↑ ↓` | 音量调节 | |
| `Space` | 暂停/播放 | |
| `P` | 视频截图 | 自动保存 |
| `I` | 画中画 | |
| `Enter` | 全屏 | |

## 二、字幕翻译功能（可选）

### 启动后端服务

```bash
cd subtitle_backend
./start.sh
```

### 使用字幕

- **快捷键**: 按 `S` 键
- **按钮**: 点击控制栏的 📝 按钮
- **配置**: 油猴菜单 → ⚙️ 字幕翻译配置

### 配置目标语言

```
油猴菜单 → ⚙️ 字幕翻译配置
→ 输入语言代码（如 en, ja, ko）
→ 确认保存
```

**常用语言代码：**

- `zh-CN` - 简体中文（默认）
- `en` - 英语
- `ja` - 日语
- `ko` - 韩语

## 三、故障排查

### 脚本未启动？

→ 刷新页面，查看控制台（F12）

### 字幕不显示？

→ 检查后端服务：`curl http://localhost:8765/health`

### 快捷键无效？

→ 确认没有输入框处于焦点状态

## 四、高级使用

### 组合技巧

**快速学习：**

```
S 键 → 字幕翻译
V 键 → 加速观看
P 键 → 截图保存
```

**高效浏览：**

```
→ 键 → 跳过无聊部分
Z 键 → 1.5倍速
I 键 → 画中画继续其他工作
```

## 五、配置选项

### 油猴菜单

- **脚本功能快捷键表** - 查看所有快捷键
- **√ 记忆播放速度** - 切换是否记住速度
- **⚙️ 字幕翻译配置** - 配置服务和语言
- **🔄 重启字幕服务** - 重启字幕识别
