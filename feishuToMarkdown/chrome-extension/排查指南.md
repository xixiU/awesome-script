# 排查步骤

请按照以下步骤排查问题：

## 1. 检查扩展是否正确加载

1. 按 F12 打开开发者工具
2. 切换到 "Console"（控制台）标签
3. 刷新页面
4. 查看是否有以下信息：
   - "飞书文档转 Markdown 插件已加载" - 表示成功
   - 或者是否有红色错误信息

## 2. 检查依赖库是否加载

在控制台输入以下命令检查：

```javascript
typeof TurndownService
typeof JSZip
```

如果返回 "undefined"，说明依赖库没有正确加载。

## 3. 检查文件结构

确认以下文件都存在：

```
chrome-extension/
├── libs/
│   ├── turndown.min.js  ← 必须存在
│   └── jszip.min.js     ← 必须存在
├── content.js
└── manifest.json
```

## 4. 检查扩展权限

1. 访问 chrome://extensions/
2. 找到"飞书文档转 Markdown"扩展
3. 点击"详细信息"
4. 确认"网站访问权限"包含飞书域名

## 5. 手动测试

在控制台输入以下代码测试：

```javascript
// 检查是否在正确的页面
console.log('当前 URL:', window.location.href);

// 检查是否能找到文档内容
console.log('文档内容:', document.querySelector('.doc-content, [data-testid="doc-content"], .suite-editor-content, .lark-doc-content'));

// 检查按钮是否存在
console.log('导出按钮:', document.getElementById('feishu-md-export-btn'));
```

## 常见问题

### 问题1: 依赖库未下载
**解决方法**: 运行 `install-deps.bat` 下载依赖库

### 问题2: 页面 URL 不匹配
**解决方法**: 检查 manifest.json 中的 matches 是否包含当前页面 URL

### 问题3: 内容脚本未注入
**解决方法**:
1. 刷新页面
2. 重新加载扩展（chrome://extensions/ → 点击刷新图标）
3. 重启浏览器

### 问题4: 选择器失效
**解决方法**: 飞书可能更新了页面结构，需要更新选择器
