## 使用方法

### 基本使用
1. 打开飞书文档页面
2. 鼠标移到右上角的"📥 导出文档"按钮
3. 选择导出格式：
   - 📥 导出 Markdown
   - 📄 导出 Word
   - ⚙️ 导出选项

### 配置导出选项
1. 鼠标悬停在"📥 导出文档"按钮上
2. 点击"⚙️ 导出选项"
3. 勾选需要导出的内容：
   - ☑️ 包含图片
   - ☑️ 包含文本
   - ☑️ 包含表格
4. 点击"保存"

## 更新步骤

1. **下载 docx 库**（如果还没下载）
   - 访问：https://cdn.jsdelivr.net/npm/docx@8.5.0/build/index.umd.min.js
   - 保存为 `libs/docx.min.js`

2. **重新加载扩展**
   - 访问 `chrome://extensions/`
   - 找到"飞书文档转 Markdown"扩展
   - 点击刷新按钮 🔄

3. **刷新文档页面**
   - 回到飞书文档页面
   - 按 F5 刷新

4. **体验新功能**
   - 现在只有一个按钮
   - 鼠标悬停展开菜单
   - 选择导出格式
