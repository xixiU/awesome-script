# 更新说明 v1.0.1

## 修复内容

### 1. 支持更多部署方式
- ✅ 飞书官方文档（feishu.cn）
- ✅ Larksuite 国际版（larksuite.com）
- ✅ 飞书私有化部署版本
- ✅ 其他自定义域名的飞书部署

### 2. 改进初始化逻辑
- 增加了更多通用的内容选择器
- 改进错误处理，即使找不到特定元素也会创建导出按钮
- 添加降级模式，确保插件在各种部署环境都能工作

### 3. 增强内容获取
- 支持更多 DOM 结构变体
- 自动过滤导航、侧边栏等不需要的内容
- 更智能的内容识别

## 如何更新

1. 访问 `chrome://extensions/`
2. 找到"文档转 Markdown"扩展
3. 点击刷新图标 🔄
4. 回到文档页面，刷新页面（F5）
5. 应该能看到右上角的"📥 导出 Markdown"按钮

## 使用说明

更新后，插件会：
1. 自动检测飞书文档（包括私有化部署）
2. 在页面右上角显示导出按钮
3. 点击按钮即可导出 Markdown 和图片

## 故障排查

如果更新后仍然看不到按钮：

1. **完全重新加载扩展**
   - 在 chrome://extensions/ 中先关闭扩展
   - 再重新开启
   - 刷新文档页面

2. **检查控制台**
   - 按 F12 打开开发者工具
   - 查看 Console 标签
   - 应该看到"文档转 Markdown 插件已加载"

3. **手动触发**
   - 如果按钮没出现，在控制台输入：
   ```javascript
   const btn = document.createElement('button');
   btn.innerHTML = '📥 导出 Markdown';
   btn.style.cssText = 'position:fixed;top:80px;right:20px;z-index:10000;padding:10px 20px;background:#0066FF;color:white;border:none;border-radius:6px;cursor:pointer;';
   document.body.appendChild(btn);
   ```
