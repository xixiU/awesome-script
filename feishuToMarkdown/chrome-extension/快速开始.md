# 快速开始

所有文件已准备就绪！按照以下步骤安装扩展：

## 1. 下载依赖库

### Windows 用户
双击运行 `install-deps.bat`

### Linux/Mac 用户
```bash
bash install-deps.sh
```

或者手动下载：
```bash
mkdir libs
curl -o libs/turndown.min.js https://cdn.jsdelivr.net/npm/turndown@7.1.2/dist/turndown.min.js
curl -o libs/jszip.min.js https://cdn.jsdelivr.net/npm/jszip@3.10.1/dist/jszip.min.js
```

## 2. 加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 开启右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `chrome-extension` 文件夹
6. 完成！

## 3. 使用扩展

1. 打开任意飞书文档页面
2. 等待页面加载完成（1-2秒）
3. 点击右上角的"📥 导出 Markdown"按钮
4. 等待处理完成
5. 自动下载 ZIP 文件

## 文件清单

⏳ libs/turndown.min.js - 需要下载
⏳ libs/jszip.min.js - 需要下载

## 故障排查

### 扩展无法加载
- 确保已下载依赖库（libs 文件夹中的两个 js 文件）
- 检查 manifest.json 格式是否正确

### 找不到导出按钮
- 刷新飞书文档页面
- 确认扩展已启用（chrome://extensions/）

### 导出失败
- 打开开发者工具（F12）查看控制台错误
- 确保文档已完全加载
