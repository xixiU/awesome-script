# 飞书文档转 Markdown v3.0 - 快速开始

## 🚀 5 分钟快速上手

### 第一步：下载依赖库

在 `libs/` 目录下需要以下文件：

1. **mammoth.browser.min.js**（新增，必须）
   - 下载：https://cdnjs.cloudflare.com/ajax/libs/mammoth/1.8.0/mammoth.browser.min.js
   - 保存为：`libs/mammoth.browser.min.js`

2. **jszip.min.js**（已有）
   - 下载：https://cdn.jsdelivr.net/npm/jszip@3.10.1/dist/jszip.min.js

3. **turndown.min.js**（已有）
   - 下载：https://cdn.jsdelivr.net/npm/turndown@7.1.2/dist/turndown.min.js

### 第二步：加载扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**
5. 选择 `chrome-extension` 文件夹
6. 完成！

### 第三步：使用扩展

#### 方式一：一键自动导出（推荐）

```
打开飞书文档页面
    ↓
等待页面完全加载
    ↓
点击右上角「📥 导出 Markdown」按钮
    ↓
扩展自动完成：
  - 🔍 查找飞书下载 API
  - 📥 自动下载 docx 文件
  - 📄 解析 docx 内容
  - 📝 转换为 Markdown
  - 📦 打包图片和文件
    ↓
自动下载 Markdown ZIP 文件
```

#### 方式二：手动上传转换（备用）

如果自动下载失败，使用此方式：

```
飞书文档页面
    ↓
点击「···」→「导出」→「导出为 Word」
    ↓
下载 docx 文件到本地
    ↓
点击浏览器扩展图标
    ↓
拖拽 docx 文件到弹窗
    ↓
点击「开始转换」
    ↓
自动下载 Markdown ZIP 文件
```

## ✅ 支持的格式

- ✅ 标题（H1-H6）
- ✅ 段落文本
- ✅ 粗体、斜体
- ✅ 有序列表、无序列表
- ✅ 代码块
- ✅ 表格
- ✅ 图片（自动提取）
- ✅ 链接

## ⚠️ 注意事项

1. **必须下载 mammoth.browser.min.js**，这是 v3.0 的核心依赖
2. **自动下载可能失败**：如果飞书 API 变化，请使用手动方式
3. **需要登录状态**：自动下载需要浏览器已登录飞书账号
4. 大文档（>5MB）可能需要较长转换时间
5. mammoth.js 专注于语义结构，不保留字体颜色等样式

## 🔧 故障排查

### 自动下载失败

提示"自动下载失败"时：
1. 使用手动方式：点击扩展图标，上传 docx
2. 检查是否已登录飞书
3. 打开 F12 查看控制台错误

### 弹窗显示"缺少依赖库"

检查 `libs/` 目录是否包含：
- mammoth.browser.min.js
- jszip.min.js
- turndown.min.js

### 转换失败

1. 打开浏览器控制台（F12）查看错误
2. 确认 docx 文件完整且未损坏
3. 尝试重新从飞书导出 docx

### 图片缺失

- 检查飞书导出的 docx 是否包含图片
- 查看控制台是否有图片提取警告
